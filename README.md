# PUTM_EV_LAP_TIMER_2022
## About this project
  Firmware for Laptimer device used on PM-08 racing car developed by PUT Motorsport. 
Written in C/C++ for STM32L4P5RET6 microprocessor, created in STM32CubeIDE 1.8.0
